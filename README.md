# WebTUI

Modular CSS Terminal-like UI library

Read the docs at https://webtui.ironclad.sh

1. Clone the repo
2. `cd` into the repo
3. Run `bun i`
5. Run `bun run dev` to start the dev server

For feature requests please open an issue.
